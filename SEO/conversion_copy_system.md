# Pet Plates — On‑Site Conversion Copy System (CTAs + Next‑Step Blocks)

Scope constraints:
- Use only the existing **10 pages** + **trust pages**.
- No new page proposals.
- One mascot owner per CTA block.

---

## 1) Universal CTA rules (sitewide)

- Keep CTAs **single-purpose** (one next step).
- Avoid medical promises.
- Use “read / learn / plan / check” verbs.
- Always include a **trust link** on YMYL pages.
